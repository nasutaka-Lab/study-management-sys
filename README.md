# 週の学習計画 (Weekly Study Planner)

スマホでの使いやすさを追求した、モダンでプレミアムな学習計画管理ツールです。PWA（Progressive Web App）に対応しており、ホーム画面に追加してネイティブアプリのように利用できます。

## ✨ 特徴

- 📱 **モバイルファースト**: スマホ片手でサクサク操作できるボタン配置とレイアウト。
- 🎨 **リッチなデザイン**: グラスモーフィズムを採用した、ダークモードベースの洗練されたUI。
- 🗓️ **曜日別管理**: 月曜日から日曜日まで、曜日ごとの学習計画を簡単に作成・管理。
- 📊 **進捗可視化**: その日の完了状況をプログレスバーでリアルタイムに表示。
- 🌙 **ダーク/ライトモード**: 好みに合わせてテーマを瞬時に切り替え可能。
- 💾 **オフライン対応 (PWA)**: サービスワーカーにより、オフライン環境でも動作します。
- 🔒 **プライバシー重視**: サーバーを介さず、データはすべてデバイス内（localStorage）に保存されます。

## 🚀 使い方

1. `index.html` をブラウザで開きます。
2. スマホで利用する場合は、GitHub Pages などにホスティングするか、ファイルをスマホへ転送してください。
3. ブラウザの共有/メニューボタンから「ホーム画面に追加」を選択することで、アプリとしてインストールできます。

## 🛠️ 技術スタック

- **HTML5**: セマンティックな構造
- **CSS3**: バニラCSS、Flexbox/Grid、アニメーション、グラスモーフィズム
- **JavaScript**: バニラJSによるステート管理とDOM操作
- **Lucide Icons**: 美しくシンプルなアイコンセット
- **PWA**: サービスワーカーとマニフェストファイルによるオフライン/インストール対応
